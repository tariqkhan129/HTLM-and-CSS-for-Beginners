Using B

This is how we make a word appear bold. Inside a product description you might see some key features in bold.

Using Strong

The use of the strong element indicates that its content has strong importance. For example, the words contained in this element might be said with strong emphasis. By default, browsers will show the contents of a strong element in bold.

Italic Using em

The em tag is used to define emphasized text. The content inside is typically displayed in italic. A screen reader will pronounce the words in with an emphasis using verbal stress
